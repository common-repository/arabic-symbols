Add Arabic Symbols, symbols and signs in quran, and special characters and numbers to the TinyMCE Editor of wordpress and remove the Latin special characters.

== Description ==

This plugin makes it possible to insert the Arabic special characters and symbols into the WordPress editor using the standard "Character Map" feature.

= Usage =
Click the **`Ω`** in the bottom row of editor buttons to view the Character Map popup window.

= Version Requirements =
This plugin requires **WordPress 4.5 or higher**.

== Installation ==

1. Go to `Plugins > Add New` in your WordPress Dashboard
2. Search for "Arabic Symbols"
3. Click "Install" for the Arabic Symbols plugin
3. Click the "Activate Plugin" link
